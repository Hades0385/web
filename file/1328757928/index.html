成績，是一切嗎?

發表於 2022-05-06 分類於 雜談 Disqus： 文章字數： 374

許多人都以成績去評判一個人，但真的是這樣嗎

一個人的好壞是以成績去判斷的嗎?拿到好成績真的重要嗎?成績代表一切嗎?我覺得不是，讀書是為誰而讀，總是會有他人為我們設下目標，但讀書不是為了自己讀嗎，拿到好成績不是靠自己的努力嗎，難道讀書是為了別人而讀嗎?成績不好就代表沒努力、沒讀書嗎，然到我考這成績是為了證明給你看的 證明給別人看的 這樣講的 讀書似乎不是為了自己 而是為了別人 而教育的主軸又是甚麼 唯有讀書高? 成績高就高品?

比較

今日我拿到了高分 我能獲得什麼 也許獲得一些人的讚賞 也許也就這樣了 常常去互相比較成績 不過這有意義嗎 你今天不管考怎樣 一定都會有人比你好 除非是最厲害的 不過最厲害也不需要被比較了 所已比較成績的意義在哪

小結

成績不代表一切，更不應該用成績去束縛一個人，去定義一個人，更不應該用來比較，一個人的人品好壞不該是以成績去做評判標準，沒有一個人是完美的。
